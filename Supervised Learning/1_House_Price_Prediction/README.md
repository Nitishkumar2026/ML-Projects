# 🏠 House Price Prediction System

A supervised machine learning project to predict residential house prices using advanced regression techniques.

## 🚀 Features
- **Random Forest Regressor** for high-accuracy predictions.
- Interactive **Streamlit Dashboard** for real-time price estimation.
- Detailed **Data Visualization** (Residuals, Correlation, Importance).

## 📊 Model Performance
- $R^2$ Score: `0.5174`
- MAE: `$163,325.48`

## 🛠 Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Train model: `python train.py`
3. Launch app: `streamlit run app.py`
