# 🎓 Student Performance Predictor

A supervised learning regression project to identify academic risk and predict final semester grades.

## 📌 Key Metrics
- **G1/G2 Scores**: Previous exams recorded.
- **Social Factors**: Alcohol consumption, family relationship, study time.
- **Target**: Final Math Grade (G3).

## 🚀 Quick Start
1. Train: `python train.py`
2. Dashboard: `streamlit run app.py`
