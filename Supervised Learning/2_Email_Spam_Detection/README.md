# 📧 Email Spam Detection System

A Natural Language Processing (NLP) tool to classify emails as Spam or Ham (Not Spam).

## 🛠 Features
- **TF-IDF Vectorization** for text processing.
- **Logistic Regression** for fast and reliable classification.
- Interactive **Streamlit App** to check custom emails.

## 📊 Stats
- Test Accuracy: `64.71%`

## 💻 Usage
1. Train: `python train.py`
2. Run app: `streamlit run app.py`
