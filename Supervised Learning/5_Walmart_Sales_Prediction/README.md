# 🛒 Walmart Sales Forecasting

Predicting weekly retail sales across stores using high-performance regression models.

## 🔥 Capabilities
- **Random Forest Model**: Captures non-linear dependencies in retail data.
- **Feature Engineering**: Incorporates CPI, Unemployment, and Holiday factors.
- **Interactive Dashboard**: Predict sales for any store and date.

## 🚀 Quick Start
1. `python train.py` to generate model.
2. `streamlit run app.py` to launch dashboard.
